### Steps to run Ollama on **macOS**
#### Step 1 : Install Ollama 
```brew install ollama```
![image](https://github.com/sudhirslab/local-llm-with-ollama/assets/160086220/0c34865f-47d1-453e-b1ce-fc61c17d1a97)

OR, directly install from the app 
Download the installer ```Ollama-darwin.zip``` from here.

<img width="252" alt="image" src="https://github.com/sudhirslab/local-llm-with-ollama/assets/160086220/830e311c-dd86-4a25-8815-cca44e495b51">

Unzip the file
  
<img width="242" alt="image" src="https://github.com/sudhirslab/local-llm-with-ollama/assets/160086220/492ccd74-ae4d-4350-aa0f-6fc86155cd68">

Double click on ```Ollama.app``` and follow the screen instruction ...

<img width="271" alt="image" src="https://github.com/sudhirslab/local-llm-with-ollama/assets/160086220/ef8d381f-468d-4ffc-a355-477ecc764ea3">

Click ```Move to Applications```

<img width="404" alt="image" src="https://github.com/sudhirslab/local-llm-with-ollama/assets/160086220/c62273bd-c68d-453b-9b8d-20005a561c07">

Press Next
  
 <img width="403" alt="image" src="https://github.com/sudhirslab/local-llm-with-ollama/assets/160086220/58d20768-690a-477b-bbf3-85e1dd1d1473">

Now Install

  <img width="409" alt="image" src="https://github.com/sudhirslab/local-llm-with-ollama/assets/160086220/622b9bc3-fccc-40d8-9c2b-91ffbaff27fa">

Now supply the credential to install

 <img width="403" alt="image" src="https://github.com/sudhirslab/local-llm-with-ollama/assets/160086220/b8545433-0399-4c0b-8d35-f3bce4f4b5b7">

Copy the command and click finish
```ollama run llama2```

Now you should see an laama icon on the task bar

 <img width="547" alt="image" src="https://github.com/sudhirslab/local-llm-with-ollama/assets/160086220/38c91c86-e148-48c6-b0c9-da7d3881bd8a">


#### Step 2 : Run Ollama 
- Open any of you favorite terminal
- Execute ```ollama run llama2```

<img width="737" alt="image" src="https://github.com/sudhirslab/local-llm-with-ollama/assets/160086220/9b6aac80-0eae-46ad-88c4-c67a96a668ed">

- Now you should see a prompt like ```>>>```
- Start typing your question.
- For example, ```who are you?``` and hit enter.
- You should see a response from Llama is something like,
```
    I am LLaMA, an AI assistant developed by Meta AI that can understand and respond to
human input in a conversational manner. I am trained on a massive dataset of text from the
internet and can generate human-like responses to a wide range of topics and questions.
I can be used to create chatbots, virtual assistants, and other applications that require
natural language understanding and generation capabilities.
```
